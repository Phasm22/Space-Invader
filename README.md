# Space Invaders Bit Game

Simple overview of use/purpose.
A simple program written in python with PyGame.

## Getting Started

### Executing program

* Create an Assests folder for the png files
* Run the main file 
* Use WASD to move and SPACE to shoot
* Step-by-step bullets
